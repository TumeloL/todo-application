# To-Do Application (Flask + SASS + SQLAlchemy)

A modern, isolated Python web application built using the Flask micro-framework, styled with Sassy CSS (SCSS), and backed by SQLAlchemy for database management. 

This repository is optimized specifically for **Windows environments**, featuring direct workarounds for common PowerShell security execution policies and asset compilation path bugs.

---

## Technical Architecture Overview

To maintain a clean system registry and ensure cross-platform compatibility, this project separates core system tasks:
- **Flask:** Drives the backend routing and renders template files.
- **SQLAlchemy ORM:** Maps database records cleanly to Python classes without requiring raw SQL strings.
- **Live Sass Compiler:** A global VS Code automation engine that watches local `.scss` code and compiles it into production-ready browser-readable `.css` files on the fly.

Step-by-Step Installation & Local Deployment (Windows)
Follow these steps sequentially inside your Windows terminal to provision and execute the application workspace:

1. Initialize Workspace Terminal
Open Visual Studio Code (VS Code) within the project folder. Launch a fresh integrated PowerShell console using the keyboard shortcut:

PowerShell
Ctrl + ` (Backtick)
2. Clear Windows Security Restrictions
By default, Windows PowerShell restricts running script execution tasks. Clear the local session lock by inputting:

PowerShell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
(Press Enter. This executes silently without text output, authorizing the local framework profile).

3. Provision and Activate the Isolation Environment
Build a isolated runtime sandbox folder, and immediately engage the context redirect switch:

PowerShell
# Create the environment folder
python -m venv .venv

# Activate the local execution loop
.\.venv\Scripts\Activate.ps1
The Golden Verification Check: Look closely at the far-left edge of your active terminal line. You must see a literal (.venv) tag prepended to the command line path before proceeding:

PowerShell
(.venv) PS C:\...\to_do_application>
4. Bulk Install Dependencies
With the (.venv) environment validation active, install the combined requirements stack inside your virtual environment using the tracking file:

PowerShell
pip install -r requirements.txt

Alternatively, if installing manually from scratch, run:
PowerShell
pip install Flask Flask-Scss Flask-SQLAlchemy

Frontend Styling Workflow (SASS Compilation)
Browsers cannot parse .scss files natively. To automate compilation without buggy Python background layers crashing over Windows backslashes, follow this setup:

Install Compiler (One-Time Setup): Open the Extensions Marketplace panel in VS Code (Ctrl + Shift + X), search for Live Sass Compiler (by Glenn Marks), and select Install.

Engage the Watch Thread: Open your static/scss/style.scss file in your editor pane. Look down at the bottom blue status bar of VS Code and click the Watch Sass icon.

The indicator will change to Watching..., instantly generating the production style.css file your layout depends on.

Linking Style to HTML Template
Ensure your templates/index.html header references the compiled asset tracking path using Jinja2 syntax exactly like this:

HTML
<link rel="stylesheet" href="{{ url_for('static', filename='scss/style.css') }}" />
Running the Application
Spin up the local development web thread:


python app.py
Hold down the Ctrl key and click the local web loopback path exposed in your terminal logs:


[http://127.0.0.1:5000/](http://127.0.0.1:5000/)
Troubleshooting: HTTP 304 Caching Issues
If you alter your style attributes in style.scss but your browser interface reflects unstyled or old changes, your browser is caching memory states (returning HTTP 304 logs).

To force a clean server state reload, focus your browser and press:


Ctrl + F5 (Hard Refresh)
Session Termination

To safely suspend execution and close the active background local development server thread, click inside the active VS Code terminal console window and input:

PowerShell
Ctrl + C
